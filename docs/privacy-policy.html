Política de Privacidade

Saiba como protegemos seus dados e informações pessoais na extensão Whats Link+.

Última Atualização: 29 de Agosto de 2025

Esta Política de Privacidade descreve como a extensão Whats Link+ - WA Easy Sender ("nós", "nosso" ou "nossa") coleta, usa, protege e compartilha seus dados. Nosso objetivo é garantir transparência e proteger sua privacidade ao permitir a funcionalidade da extensão.

1. Dados Coletados

Coletamos apenas o necessário para o funcionamento da extensão. Todos os dados são armazenados localmente no seu navegador.

  • Configurações de Usuário: Seu código de país padrão (ex: "+55") e serviço preferido ("App" ou "Web"). Isso é necessário para configurar a extensão às suas preferências.
  • Números Recentes: Os últimos cinco números de telefone que você usou com a extensão. Isso é usado para fornecer atalhos de acesso rápido.
  • Dados Técnicos: Não coletamos dados técnicos como versão do navegador, endereço IP ou logs de erro.

Não coletamos informações pessoais (ex: nome ou email), pois a extensão não possui contas de usuário ou análises.

2. Como Usamos Seus Dados

Seus dados são usados exclusivamente para:

  • Configurar as configurações padrão da extensão
  • Preencher a lista de "Números Recentes" para sua conveniência
  • Gerar links e QR codes com base na sua entrada

Não usamos seus dados para anúncios ou vendemos para ninguém.

3. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros.

  • Com WhatsApp/Meta: A única comunicação externa ocorre quando você inicia uma ação (como clicar em "Enviar Mensagem" ou em um link gerado), que envia o número de telefone e mensagem opcional para os servidores oficiais do WhatsApp. Esta é uma ação iniciada pelo usuário e essencial para o funcionamento do serviço.

Nenhuma outra parte recebe seus dados - sem análises, sem anunciantes.

4. Segurança dos Dados

Protegemos seus dados com:

  • Armazenamento Local Seguro: Todas as configurações e números recentes são armazenados de forma segura na área chrome.storage.sync do seu navegador.
  • Processamento em Memória: Os números de telefone e mensagens que você insere são processados no momento para gerar links e são não armazenados, exceto para o recurso "Números Recentes".

Nenhum sistema é 100% seguro, mas tomamos todas as medidas razoáveis para manter seus dados seguros no ambiente seguro do seu navegador.

5. Atividade de Navegação Web

A extensão não monitora, acessa ou rastreia sua atividade de navegação web em nenhum site. Ela age apenas em números de telefone que você seleciona manualmente ou digita no popup.

6. Seus Direitos

Você tem controle total sobre seus dados:

  • Gerenciar: Você pode visualizar e alterar suas configurações padrão a qualquer momento no popup da extensão.
  • Excluir: Você pode limpar seus "Números Recentes" e configurações ao limpar os dados da extensão do seu navegador ou reinstalando a extensão.
  • Parar: Desinstale a extensão para encerrar toda coleta e remoção de dados armazenados.

7. Retenção de Dados

Mantemos dados apenas enquanto a extensão estiver instalada:

  • Configurações e Números Recentes: Armazenados enquanto a extensão estiver instalada, ou até você limpar os dados.
  • Dados de Entrada: Os números de telefone e mensagens são processados no momento e não são retidos, exceto para a lista "Números Recentes".

8. Conformidade de Uso Limitado

O uso das informações pela extensão adere às regras de Uso Limitado da Chrome Web Store:

  • Os dados são usados apenas para fornecer e melhorar os recursos voltados para o usuário da extensão.
  • Os dados nunca são vendidos a terceiros, usados para publicidade ou para fins relacionados a crédito.
  • Os dados nunca são transferidos para outras partes, exceto quando iniciado pelo usuário para abrir um chat nos domínios oficiais do WhatsApp.

9. Atualizações da Política

Podemos revisar esta política conforme necessário. Postaremos quaisquer atualizações aqui, vinculadas no popup da extensão e na página da Chrome Web Store.

Política de Privacidade do Whats Link+

Última Atualização: 29 de Agosto de 2025

1. Dados Coletados

Coletamos apenas o necessário para o funcionamento da extensão. Todos os dados são armazenados localmente no seu navegador.

  • Configurações de Usuário: Seu código de país padrão (ex: "+55") e serviço preferido ("App" ou "Web").
  • Números Recentes: Os últimos cinco números de telefone que você usou na extensão.
  • Dados Técnicos: Não coletamos dados técnicos como versão do navegador, endereço IP ou logs de erro.

2. Como Usamos Seus Dados

Seus dados são usados exclusivamente para:

  • Configurar as configurações padrão da extensão
  • Preencher a lista de "Números Recentes"
  • Gerar links e QR codes com base na sua entrada

3. Compartilhamento de Dados

Não compartilhamos seus dados com terceiros. A única comunicação externa ocorre quando você inicia uma ação no WhatsApp.

4. Segurança dos Dados

Protegemos seus dados com:

  • Armazenamento local seguro
  • Processamento em memória

5. Direitos do Usuário

Você tem controle total sobre seus dados:

  • Gerenciar configurações a qualquer momento
  • Excluir dados recentes
  • Parar o uso desinstalando a extensão

Ao usar Whats Link+, você aceita esta política.
